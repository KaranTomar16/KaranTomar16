# Assets Folder

This folder contains all the media assets used in the main README.md file.

## Current Assets:
- `dance_ascii.gif` - Animated GIF displayed in the profile (add this file when ready)

## How to Add Assets:
1. Place your images, GIFs, or other media files in this folder
2. Reference them in README.md using relative paths like: `./assets/your-file.gif`
3. Commit and push the assets along with your README updates

## Recommended Assets to Add:
- Profile GIFs or animations
- Project screenshots
- Custom badges or icons
- Banner images
